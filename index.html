Authentication

This is the documentation of the authenticaton functionality. The following resources will allow you to manage the access credentials of users and use these for accessing restricted resources. Users of the API first need to register. Once a user is successfully registered he can acquire a token which will allow her to access restricted parts of the API.

Authentication

Register a new user

Registers a new user by email address. As opposed to Facebook registration/login where all data is automatically fetched from Facebook, the client must provide the user's data.
